# Truck_of_Twitter_Messages
为了定期获取图形学与三维视觉国外学术界与工业界同行在社交媒体上的高质量分享，了解他们分享的最新成果、观点讨论以及课程预告等内容，中科大[张举勇](http://staff.ustc.edu.cn/~juyong/)老师课题组 (USTC-3DV) 的同学们维护了这样一个半月刊栏目。涉及的领域包括但不限于建模、渲染、动画和物理模拟。相关的推文和讨论可在[Twitter List](https://twitter.com/i/lists/1425000300553728001?s=20)查阅到，同时关注列表也会被不断扩充。欢迎大家在Issues里为这个项目给予宝贵的意见及支持！
